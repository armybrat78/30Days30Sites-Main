Bot Projects

Includes Chatbots built for Slack and Facebook.

1. Slack and Study

This was a virtual collaboration project at Chingu Cohorts. Slack and Study is a Slack Slash command app to manage private groups in a Slack Team. It is extensively being used in the Chingu Cohorts for managing private study groups and project teams.

2. Intro-bot

This is another Chingu collaboration project that is currently in progress. Considering the large number of people at Chingu, we decided to build a Slack bot that stores a brief introduction of members so that we can know about any team-mate by simply asking the bot about them instead of searching through the channels for the intro-message.

We also wanted to keep a track of the most active, useful and helpful members in the group. So, the bot uses NLP tools to extract such information from the messages and store them as points for each member.
